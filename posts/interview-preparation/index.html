  • [x] identify and resolve memory leak? Instruments, persistent vs transient objects, filtering for your custom data types, how to ensure the leak is gone? -> push and pop VC 10 times and memory level remains constant -> Persistent objects are using up memory; transient objects are not. -> Transient objects already deallocated
  • [x] How to list files in a directory? FileManager -> sandboxing -> increase security by limiting what an app has access to -> app can’t access another apps sandbox -> e.g. photo library access only through system interface (permission dialog)
  • [x] how many bits are used to store data: Float -> 32 bit, Double -> 64 bit, CGFloat -> 32 (watch) or -> 64 bit depending on device
  • [x] Float or Double? -> Double is default -> higher precision, when using Float -> CGFloat is more common
  • [x] how to cache data in memory? -> NSCache -> gets cleared by the system when memory is low -> explain choice and why I like it -> Code: https://www.hackingwithswift.com/example-code/system/how-to-cache-data-using-nscache
  • [x] KVO: helpful to watch for changes on values where there is no delegate -> WKWebView -> similar to willSet / didSet, but outside of type definition, relies on ObjC runtime (@objc and NSObject) -> Apple makes no guarantee about UIKit remaining KVO-compatible -> https://www.hackingwithswift.com/example-code/wkwebview/how-to-monitor-wkwebview-page-load-progress-using-key-value-observing -> https://www.hackingwithswift.com/example-code/language/what-is-key-value-observing
  • [x] Purpose of CADisplayLink? -> attach code to the user interface drawing loop so that your code always gets called immediately after a frame has been drawn and you have maximum time available to you -> refresh UI when screen is redrawn (every 60th second)
  • [x] escaping vs non escaping closures: escaping when a closure might be called after the function has returned (e.g. after a delay) -> @escaping when asynchronous completion
  • [x] Core Image: less used than Core Graphics, filtering images (blurring / sharpening, adjusting colors)
  • [x] Core Graphics: drawing basic shapes or text and resizing images
  • [x] defer: delay a piece of work until a method ends -> saving a file once everything has been written -> many guard statements and not want to copy code in the return statements -> for cleanup or resetting state
  • [x] ReplayKit? record, save and broadcast user activity -> games or submitting error report videos
  • [x] SpriteKit vs SceneKit? SpriteKit 2D, SceneKit 3D, both on top of Metal -> can mix both
  • [x] UIMenuController? interact with textfields like Cut and Paste -> own actions -> discoverability of functionality? -> never used, but canPerformAction can be overwritten to prevent paste actions
  • [x] UIKit least familiar: UIPageViewController / UIVisualEffectView aren’t used so often -> UIKit dynamics (physic based animations) -> Drag / Drop -> Pencil Interactions
